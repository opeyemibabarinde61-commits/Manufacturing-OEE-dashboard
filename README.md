# Manufacturing OEE Dashboard

## Project Overview

This project demonstrates the development of an interactive Manufacturing Overall Equipment Effectiveness (OEE) dashboard using Microsoft Power BI.

The dashboard monitors production performance for four manufacturing systems:

- Monogram Machine
- DTF Line
- DTG Printer
- Robotic Knitting Machine

It combines production data and downtime records to calculate key manufacturing KPIs and provide actionable insights for production improvement.

---

## Objectives

- Monitor production performance
- Analyse machine downtime
- Calculate OEE
- Identify major downtime causes
- Support continuous improvement decisions

---

## Key Performance Indicators

- Availability
- Performance
- Quality
- Overall Equipment Effectiveness (OEE)
- Planned Production Time
- Total Downtime
- Total Production
- Good Products

---

## Dashboard Features

- Machine downtime analysis
- Downtime category breakdown
- OEE KPI cards
- Availability, Performance and Quality metrics
- Interactive machine filtering
- Manufacturing performance dashboard
